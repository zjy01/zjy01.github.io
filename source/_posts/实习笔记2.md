### 事务
今天主要是看Slim框架，slim是一个轻量级的php框架，比较适合写接口。

### 收获
当然是新认识了一个框架啦，还学会了启动php内置服务器。

1. 启动Php内置服务器
```bash
php -S 0.0.0.0:8088 [dir_root] [router]

```

2. Slim在apache服务器的配置
slim使用静态路由，自带了一个.htaccess文件重写路由规则，
所以需要开启 mod_rewrite模块，同时修改httpd.conf的AllowOverride权限
```bash
#开启write模块
sudo a2enmod write

#修改apache配置[httpd.conf]，使得相应的Directory 的 AllowOverride为all
sudo vi /etc/apache2/apache2.conf

```
如下
```
<Directory />
           Options FollowSymLinks
           #AllowOverride None
           #Require all denied
           AllowOverride All
           Require all granted
   </Directory>

   <Directory /usr/share>
           AllowOverride None
           Require all granted
   </Directory>

   <Directory /var/www/>
           Options Indexes FollowSymLinks
           AllowOverride All
           #None
           Require all granted
   </Directory>


```
